启用后，Groovy 脚本将在 Jenkins 脚本安全沙箱中执行。 这将限制对潜在危险操作的访问以保障安全。

沙箱限制的操作：

  • 文件系统访问（读写文件）
  • 网络操作（HTTP 请求、套接字）
  • 进程执行（运行外部命令）
  • 直接访问 Jenkins 内部 API

沙箱中允许的操作：

  • 访问任务、构建和参数信息
  • 字符串操作和基本 Groovy 操作
  • 已批准的方法调用（由管理员白名单授权）

建议：除非您信任脚本作者并且需要访问受限操作，否则请启用此选项。 需要受限操作的脚本将失败，并显示明确的错误消息，指出被拒绝的方法。

管理员可以通过 Jenkins → 脚本控制台或在系统配置中配置脚本批准来批准特定的方法调用。

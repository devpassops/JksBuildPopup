Groovy script executed before each build. The script has access to the following bindings:

  • job - The current Jenkins job
  • jenkins - The Jenkins instance
  • env - Environment variables map
  • params - Build parameters map
  • build / currentBuild - The current build (if available)

Build parameters are also available as direct variables. For example, if you have a parameter named DEPLOY_ENV, you can access it as DEPLOY_ENV or params.get('DEPLOY_ENV').

The script must return a Map with the following keys:

  • blockBuild (Boolean, default: false) - If true, block the build and only show the popup
  • showPopup (Boolean, default: false) - If true, show a popup notification
  • popupContent (String) - The message to display in the popup
  • popupTitle (String, default: "Build Notification") - The title of the popup

Example: 

if (job.isBuilding()) {
    return [blockBuild: true, showPopup: true, 
            popupContent: "Job ${job.name} is already building!", 
            popupTitle: "Build Warning"]
} else {
    return [blockBuild: false, showPopup: false, popupContent: '']
}

Note: Use map literal syntax [key: value], not method calls like popupContent('xxx').
